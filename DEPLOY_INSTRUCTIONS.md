# Deploy Your SpendSmart Website

## Quick Deploy Options

### Option 1: Netlify (Easiest - 2 minutes)

1. Go to https://netlify.com
2. Sign up (free, can use GitHub)
3. Drag the `website` folder onto Netlify
4. Get instant URL: `https://random-name.netlify.app`
5. Done! ✅

**Your URL will be something like:** `https://spendsmart-app.netlify.app`

### Option 2: GitHub Pages (Free Forever)

1. Create a new GitHub repository: `spendsmart-website`
2. Upload the `website` folder contents to the repo
3. Go to Settings → Pages
4. Select "main" branch, "/ (root)" folder
5. Your site: `https://yourusername.github.io/spendsmart-website`

### Option 3: Vercel (Fast & Easy)

1. Go to https://vercel.com
2. Sign up with GitHub
3. Import repository or drag & drop the `website` folder
4. Instant deployment

## Custom Domain (Optional)

All these services let you add a custom domain for free:
- `spendsmart.app` (if available)
- `spendsmart.io`
- `getspendsmart.com`

## What You Need for App Store

- ✅ Website URL (from above)
- ✅ Privacy Policy URL (your-site.com/privacy-policy.html)
- ✅ App Icon (120x120 to 1024x1024 PNG)

## Next Steps

1. Deploy the website using one of the options above
2. Update the email in `privacy-policy.html` with your real email
3. Add App Store badge when your app is approved
4. Use the URL for App Store submission
